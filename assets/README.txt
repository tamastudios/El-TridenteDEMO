EL TRIDENTE · LOGO PACK WEB
===========================

Contenido principal:
- icon/: logo solo en PNG transparente y con fondo negro, en 256/512/1024/2048 px.
- favicon/: favicons PNG, favicon.ico, apple-touch-icon y manifest para web.
- header/: versiones horizontales para cabecera/nav.
- stacked/: versiones apiladas, icono arriba + texto.
- wordmark/: texto solo.
- social/: imagen Open Graph 1200x630 para compartir en redes.
- svg/: icono en SVG con fill=currentColor para poder colorearlo por CSS.
- preview/: vista rápida del pack.

Colores incluidos:
- Oro: #FFD218
- Negro: #111111
- Blanco: #FFFFFF

Recomendación web:
<link rel="icon" href="/favicon/favicon.ico" sizes="any">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon/favicon-32x32.png">
<link rel="apple-touch-icon" sizes="180x180" href="/favicon/apple-touch-icon.png">
<link rel="manifest" href="/favicon/site.webmanifest">

Para la cabecera de una web con fondo oscuro:
/header/logo-header-gold-transparent.png

Para el logo solo en alta calidad:
/icon/logo-icon-gold-transparent-2048.png

Nota:
El símbolo se ha limpiado a partir del archivo blanco/negro que enviaste. El texto del header/wordmark está reconstruido con una tipografía serif de sistema; si quieres una réplica exacta del rótulo original, haría falta el archivo vectorial original o el nombre de la tipografía.
